# Creare un fumetto

## Generazione script

``` text
Agisci come se fossi un disegnatore Marvel.
Crea lo script per un fumetto utilizzando le informazioni presenti nel file.
Il fumetto e' ambientato in una classe di una scuola elementare della provincia di Salerno in Campania e ha i seguenti 3 personaggi come protagonisti del fumetto: 
- Carla: e' una insegnante di scuola elementare,
- Maria: e' una bambina di 8 anni molto curiosa
- Massimo: e' un bambino, saputello, di 8 anni. 
Estendi le caratteristiche dei personaggi coinvolti aggiungendo caratteristiche fisiche e abbigliamento. 
Il fumetto deve essere composto di massimo 10 scene e deve contenere un linguaggio semplice adatto a studenti delle scuole medie inferiori.
```

E' possibile aggiungere i personaggi che si desidera e il livello di linguaggio.

## Immagini dei personaggi

``` text
Crea una descrizione dettagliata di Massimo da poter utilizzare per la creazione dell'immagine.
```

``` text
Crea l'illustrazione di Massimo come se fossi un disegnatore della Marvel.
<descrizione dettagliata di Massimo>
Usa uno sfondo trasparente per garantire che l'attenzione resti interamente concentrata sulla posa dinamica e sulla presenza del personaggio.
```

## Immagini per le scene

``` text
Crea una descrizione dettagliata della scena 1 da poter utilizzare per la creazione dell'immagine.
```

``` text
Crea lo sfondo per la scena numero 1 come se fossi un designer della Marvel.
<descrizione della stanza>
La scena deve contenere solo la stanza in cui si svolge la scena e non deve contenere i personaggi che verranno aggiunti in post-produzione.
```

## Titolo del fumetto

``` text
Crea un titolo in stile Marvel per la prima scena del fumetto. Il titolo è "<titolo del fumetto>". Crea un titolo simile a quello di una copertina di un fumetto. Disegna il titolo non necessariamente in orizzontale, puoi anche disegnarlo in diagonale o curvo. Lo sfondo è trasparente, per garantire che l'attenzione rimanga interamente sul titolo. Il testo deve essere sempre scritto correttamente.
```

## Quiz sul fumetto

``` text
Crea 5 domande e risposte relative al fumetto. Gli studenti devono poter rispondere alle domande dopo aver letto il fumetto quindi le domande essere costruite in modo che la risposta si possa trovare leggendo il fumetto. Fornisci più risposte per ogni domanda e fornisci la risposta corretta.
```
