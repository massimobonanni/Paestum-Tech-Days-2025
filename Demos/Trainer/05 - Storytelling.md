# Storytelling

``` text
Scrivi una storia per una classe di bambini delle scuole elementari che racconti in maniera semplice l'argomento contenuto nel file allegato. 
Inserisci i seguenti personaggi:
- Un vecchio saggio greco che racconta la storia
- Amanda una bambina curiosa che fa tante domande
- Massimo il saputello della classe
Usa un linguaggio semplice e facilmente comprensibile da bambini delle scuole elementari. Non usare parole sconvenienti, di cattivo gusto, parolacce o con significato negativo. 
```
