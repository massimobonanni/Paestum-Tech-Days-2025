# Storytelling

```
Scrivi una storia per una classe di bambini delle scuole elementari che spieghi in maniera semplice l'argomento del file allegato. 
Inserisci i seguenti personaggi:
- Giuseppe che racconta la storia
- Amanda una bambina curiosa che fa tante domande
- Massimo il saputello della classe
Usa un linguaggio semplice e facilmente comprensibile da bambini delle scuole elementari. Evita di usare parole sconvenienti o con significato negativo. 
```
